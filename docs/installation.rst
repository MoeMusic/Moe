Installation
############

.. _Installation Guide:

**********
Installing
**********

#. `Install pipx <https://pypa.github.io/pipx/installation/>`_
#. Install moe

    .. code-block:: bash

        $ pipx install moe

That's it! Check out the :doc:`Getting Started <getting_started>` page to start using Moe.

*********
Upgrading
*********

    .. code-block:: bash

        $ pipx upgrade moe

************
Uninstalling
************

    .. code-block:: bash

        $ pipx uninstall moe
